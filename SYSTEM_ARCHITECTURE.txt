└──────────────────────────────────────────────────────────────────────────────┘

   TRADITIONAL RAG                         KNOWLEDGE GRAPH RAG
   ═══════════════                         ═══════════════════

   Text chunks only              →         Relationship nodes
   Single-hop retrieval          →         Multi-hop traversal
   No medical validation         →         Procedure-diagnosis matching
   Can't detect networks         →         Provider network detection
   Black box results             →         Complete explainability
   65-75% accuracy               →         87-92% accuracy
   15-25% false positives        →         4-8% false positives

   EXAMPLE OUTPUT:                         EXAMPLE OUTPUT:
   ───────────────                         ───────────────
   "Found 3 cardiology claims"             "3 high-risk claims detected:
                                           - Duplicate billing pattern
                                           - Provider fraud history: 5 incidents
                                           - Diagnosis mismatch (cardiac/migraine)
                                           - Network: 3 connected providers
                                           - Recommendation: Deny & investigate"
